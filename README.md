# SubTrackr

Eine moderne Abo-Verwaltungs-App, entwickelt mit React und Supabase.

## Schnellstart

### Voraussetzungen
- Node.js (v18 oder höher)
- npm oder yarn

### Installation

```bash
# Dependencies installieren
npm install

# .env Datei erstellen
cp .env.example .env
# Füge deine Supabase-Credentials in die .env ein
```

### Entwicklung starten

```bash
npm run dev
```

Die App läuft dann auf `http://localhost:5173`

### Build für Produktion

```bash
npm run build
```

### Preview des Production Builds

```bash
npm run preview
```

## Tech-Stack

- **React 18** - UI Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Supabase** - Backend & Datenbank
- **Lucide Icons** - Icons
